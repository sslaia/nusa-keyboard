# Nusa Keyboard

Nusa Keyboard is an Android keyboard application designed to support Wikipedia volunteers and language communities in Indonesia. Rebranded from Nias Keyboard, it is built to be accessible to all wiki volunteers across Indonesia by offering localized layouts for both **Bahasa Indonesia** and regional languages (starting with **Nias / Li Niha**).

## Features

- **Dual-Language Layout Support**: Seamlessly type in **Bahasa Indonesia** or **Nias (Li Niha)**.
- **Dynamic Layout Switching**: Swap layouts instantly using the globe (`🌐`) key.
- **Typing Language Selector**: Pick your preferred default typing layout directly from the app Settings.
- **Wikipedia Code Bar (Wiki Mode)**: A quick-access toolbar for Wikipedia syntax shortcuts (e.g., `[[`, `]]`, `{{`, `}}`, `'''`) designed specifically to boost editing productivity for wiki contributors.
- **Intelligent Auto-suggestions & Dictionary Learning**: Suggests vocabulary for both languages and learns new words as you type.
- **Special Characters on Long Press**: Easily access regional characters like `ŵ` and `ö` or accented variants.
- **Dark Mode**: Fully supports night theme and Gboard-like system colors.

## Getting Started

1. **Enable the Keyboard**: Open the Nusa Keyboard app, tap **Activate Keyboard**, and toggle on **Nusa Keyboard** in your Android Language & Input settings.
2. **Select Input Method**: Tap any text field, select the keyboard settings icon on your device, and set the default input method to **Nusa Keyboard**.
3. **Configure Defaults**: Open the Nusa Keyboard settings app to choose your default typing language (Nias or Indonesian) and enable/disable features like Dark Mode and Wiki Mode.
